GitHub Copilot Labs — Get Explanations and Translate Code

January 10 2024
Programming Github AI
Written by
Sakib Sami

If you're a programmer, you most probably know about or have heard of GitHub Copilot, the incredibly powerful AI autocomplete tool. But there are a couple of experimental features of Copilot that come with the “Github Copilot Labs” extension that you may not know about that are very fascinating and useful. Let's have a quick look at them.


How To Get It?

GitHub Copilot Labs is separate from but dependent on the GitHub Copilot extension and is currently available only for VS Code. You can download the Copilot Labs extension through the VS Code Marketplace, just like you would.


The “Explain” Feature

Is there a block of code that you just can't fully grasp? Or some code in a language you don't know but need to understand? The Explain feature of Copilot is just what you need in such situations. Using this feature you can get natural language (English) explanations of any code snippet! To do so, highlight the block of code you want the explanation for, choose one of the options for the type of explanation from the dropdown menu, and then click “Ask Copilot” under the “explain” tab to get the explanation.




The “Language Translation” Feature

Ever needed to translate your code from one coding language into another? You can do that very easily with the Language Translation feature of GitHub Copilot Labs. Just highlight the block of code you want to translate, select your desired language, and then click “Ask Copilot” under the “language translation” tab to get the translation.
